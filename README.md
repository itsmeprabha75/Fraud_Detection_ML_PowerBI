# 💳 Bank Transaction Fraud Detection using Machine Learning & Power BI

## 📌 Project Overview

Financial fraud has become one of the biggest challenges in digital banking. This project focuses on detecting fraudulent banking transactions by combining machine learning techniques with interactive Power BI dashboards to help financial institutions identify suspicious activities and support faster decision-making.

The project covers the complete analytics lifecycle, including data preprocessing, exploratory data analysis (EDA), machine learning model development, model evaluation, and business intelligence reporting.

---

## 🎯 Objectives

- Detect fraudulent banking transactions using machine learning.
- Analyze customer and transaction behavior.
- Identify fraud patterns across locations, devices, merchants, and transaction types.
- Build interactive dashboards for fraud monitoring.
- Support data-driven fraud prevention strategies.
